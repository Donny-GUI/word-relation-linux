# word-relation-linux
Relate to topics to one another

# Installation

LINUX/MACOS only

```bash
git clone https://github.com/Donny-GUI/word-relation-linux.git
cd word-relation-linux
python3 main.py "Super Mario Bros" "Super Mario 64"

```
